# 🎯 Multi-Face Attendance System with Real-Time Notification

## 📌 Overview
This project detects multiple faces and marks attendance automatically.
It also sends real-time notifications for absentees.

---

## 🚀 Features
- Multi-face detection
- Automatic attendance marking
- Email notifications
- Flask web dashboard

---

## 🛠️ Technologies Used
- Python
- Flask
- OpenCV
- SQLite

---

## ▶️ How to Run

1. Clone the repository  
2. Install dependencies  
3. Run `app.py`  
4. Open browser: http://127.0.0.1:5000
